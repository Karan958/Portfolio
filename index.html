Intelligent Tic-Tac-Toe Game

This is a simple implementation of the classic game Tic-Tac-Toe, where the computer player uses the Minimax algorithm to make intelligent moves.In this Tic-Tac-Toe game, the 3x3 grid is used, with 'X' representing the human player and 'O' representing the computer player. The human player always goes first and selects a cell by entering row and column numbers. The game continues with alternating turns between the human and computer players until there is a winner or a draw.
